User permissions are managed by an administrator in the [edit users](./edit-users) window.

Users may be assigned the **Owner** role or a **User** role.

### Permissions

The permissions for each role are: 

|Permissions|Owner|User|
|----|----|----|
|Sign-up for a Rita Go instance|✓| |
|Add users | ✓ | |
|Edit user permissions | ✓ | |
|Add knowledge articles| ✓ | |
|Maintain knowledge articles | ✓ | |
|View knowledge articles | ✓| |
|Add external connections| ✓ |  |
|Chat with Rita Go| ✓ | ✓ |
