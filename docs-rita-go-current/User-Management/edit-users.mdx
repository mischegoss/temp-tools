On the **Users** dashboard, you can edit or delete existing users.

### Edit Existing Users
To edit a user's name, email or role:

1. Navigate to the user's name in the users table.
2. Click the three-dot icon at the end of the user's row.
3. Click **Edit**.
4. In the *Edit user* window, change *Name*, *Email*, or *Role*.
5. Click **Update** in the bottom right corner.

### Delete Existing Users
To delete a user:

1. Click the checkbox next to the user's name.
2. Click the red **Delete** button that appears when the name is checked.
