To get started using Rita Go, add at least one Knowledge Article to create a knowledge base for your instance.

On the dashboard:
1. Click the **+** at the top right near *Knowledge Articles*.
2. In the window that opens, select one or more files from your device. Supported file types include **.pdf, .csv, .txt,** and **.doc**.
3. Click **Open**.
4. In the right panel under *Knowledge Articles*, click the name of the article you uploaded.
5. Confirm the *Status*.
6. Return to the main screen or click **New Chat** once the article is **Processed**.

For more detailed information about best practices for knowledge articles, see [Knowledge Base Setup](../Knowledge-Base-Management/knowledge-base-setup). For more information about uploading additional articles, see [Uploading Knowledge Articles](../Knowledge-Base-Management/uploading-knowledge-articles).
