The Barista - ServiceNow integration allows you to connect your Barista tenant to ServiceNow for user synchronization and integration configuration.

This guide walks you through setting up the connection, including configuring credentials, enabling the integration, and defining user provisioning criteria.

By the end of this guide, you will have:

- Connected Barista to ServiceNow
- Configured the required integration settings
- Defined which users are synchronized

:::note

This setup is typically performed by an administrator with access to both Barista and ServiceNow.

:::
