**Issue:** When I run the installer, I get a message that a technical error has occurred, and I’m advised to try again.

**Possible resolution:** Verify that you can connect from your local computer to your cloud tenant before retrying. If you still receive the same error after that, it probably means that the installer can’t connect to the cloud environment for another reason.

**Issue:** When I run the installer I get a message that one or more of the connection details are incorrect.

**Possible Resolution:** Double-check that you have correctly entered all the values that you acquired in [Acquiring API Access Details](../Getting-Started/Setting-Up-Hybrid-Components/acquiring-api-access-details.mdx) before trying again.

**Issue:** I am unable to successfully connect to the cloud environment and execute workflows even though my Internet connection is fine.

**Possible Resolution:** In your firewall, open port 443 for communication with `https://bridge.resolveactions.com`.
