title: New Workflows
---
The **Create New** option adds a blank workflow to the Designer canvas, represented by a Start and End element.

To begin, enter a name and (optionally) a description. Providing a brief summary is recommended to help others understand the workflow's purpose. See [Create a New Workflow](../Product-Navigation/Workflow-Designer/Workflow-File-Options/create-new-workflow.mdx) for step-by-step instructions.

Assigning one or more tags is also recommended. Tags act as keywords to help organize and locate workflows easily.
