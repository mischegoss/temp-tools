title: Add Tags to Workflow
---
Tags are keywords used to organize workflows and make them easier to find. They typically represent actions or categories relevant to the workflow—for example, tags for a workflow like *Monitor Windows*, *Event Log* might include *Monitoring*, *Windows*, and *Logs*.

The Workflow Designer provides a selection of preconfigured tags, but you can also create and add custom tags as needed.

For step-by-step instructions see [Add Tags](../Product-Navigation/Workflow-Designer/Workflow-File-Options/add-tags.mdx).
