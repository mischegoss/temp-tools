## Activity Description

Sends an SNMP trap containing a specific SNMP object assigned with a value.

## Output

The SNMP object value.

## Settings


* **OId** – The object identifier assigned to the individual object within the Management Information Base (MIB).  
   Example: `1.3.6.1.4.868.2.4.1.2.1.1.1.3.3562.3`
* **Value** – The value assigned to the SNMP object.  
   Example: `123456`
* **Host Name** – The device running SNMP.  
   Example: `192.168.1.106`
* **SNMPv1**/**SNMPv2c**/**SNMPv3** – The SNMP version that the device is running.
* **v3 Auth/Priv** – If the device is running SNMPv3, type the Engine Id, User, Auth Password, and Enc Password.
