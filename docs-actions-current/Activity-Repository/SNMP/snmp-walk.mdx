## Activity Description

Retrieves a list of Object IDs of a specified device, based on a given OID range.

## Settings


* **Old** – The object identifier assigned to the individual object within the Management Information Base (MIB).  
   Example: `1.3.6.1.4.868.2.4.1.2.1.1.1.3.3562.3`
* **Host Name** – The device running the SNMP.  
   Example: `192.168.1.106`
* **Community** – The group to which the device running SNMP belongs.  
   Example: `public`
* **Limit SNMP Walk Result** – Select to limit the SNMP Walk Result.
* **Walk Limit** – Type the desired Walk Limit.
* **SNMPv1**/**SNMPv2c**/**SNMPv3** – The SNMP version that the device is running.
* **v3 Auth/Priv** – If the device is running SNMPv3, type the Engine Id, User, Auth Password, and Enc Password.
