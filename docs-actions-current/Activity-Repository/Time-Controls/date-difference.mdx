## Activity Description

Calculates the difference (in a selected time volume) between two time stamps.

* **Use Current date as first date** – Calculates the difference between today's date and a second date.
* **First Date Format** – The first date's format (in case the current date is not selected).
* **First Date** The first date (in case the current date is not selected).
* **Second Date Format** – The second date's format (e.g., "M/d/yyyy h:mm:ss tt").
* **Second Date** – The second date (%FileModifiedDate%).
* **Return Format** – The time volume in which the result is calculated (seconds, minutes, hours, days, months, or years).
