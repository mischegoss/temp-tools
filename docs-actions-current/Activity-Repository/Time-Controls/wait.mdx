## Activity Description

Pauses the workflow for a specified period of time.

## Settings

* **Default Time Format/Custom Time Format** – Determines whether to use a constant time value or a variable. Default Time Format uses the timeout listed in the activity; by default this is one minute for most activities.
* **Time Interval** – If you elected to use a variable time value (custom time format), set the time unit that the variable represents.
* **Time** – The name of the variable holding a numeric value.  
   Example: `%counter%`
