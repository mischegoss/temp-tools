## Activity Description

Converts a date to a specified target format.

* **Input Time** – The source date (e.g., `%Date%`).
* **Source Format** – The format of the source date (e.g., `MM/dd/yyyy`).
* **Target Format** – The format of the target date (e.g., `MM_dd_yyyy_HH_mm`).
