## Activity Description

Adds an interval to a given date.

* **Input Time** – The given date (e.g., %Date%).
* **Source Format** – The format of the given date (e.g., MM/dd/yyyy).
* **Interval Type** – The interval type to be added (Seconds/Minutes/Hours/Days/Months/Years).
* **Interval** – The value of the selected interval.
* **Target Format** – The format of the target date (e.g., MM/dd/yyyy).
