## Activity Description

Starts a Telnet session.

## Settings


* **Host Name** – The host name or IP address.
* **Port** – The Telnet session port (e.g., 23).
* **Wait For String** – The Telnet command prompt (e.g., `[root@localhost~]#`).
* **Session Maximum Time** – The length of the session (e.g., 00:10:00).
