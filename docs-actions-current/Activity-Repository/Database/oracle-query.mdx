## Activity Description

Executes an Oracle SELECT query on an Oracle database and returns a table.

## Output

A ResultSet (table).

:::note
To use Oracle activities, you need to install the full administrator package of [Oracle Client](https://www.oracle.com/database/technologies/instant-client/winx64-64-downloads.html) on the machine where the VAR::PRODUCT_FULL executor is installed.  
After installing the Oracle Client, contact your Resolve representative for updating the path to the client in the activity's configuration.
:::

## Settings

* **Query** – The query's text.
* **Connection String** – Specifies the required connection string in order to connect the database.  
  For example:
  ```sql
  Data Source=(DESCRIPTION=(ADDRESS_LIST=(ADDRESS=(PROTOCOL=TCP)(HOST=MyHost)(PORT=MyPort))(CONNECT_DATA=(SERVER=DEDICATED)(SERVICE_NAME=MyOracleSID)));User Id=myUsername;Password=myPassword
  ```
* **User Authentication** – Determines whether a user's authentication credentials are required to execute the query (if the credentials are not specified in the clear text connection string).
* **User name** – An Oracle user authorized to execute the query (e.g., `oradba`).
* **Password** – The user's password.

:::note
To avoid translation of `%TEXT%` (in case TEXT is not a variable name), add `@` before the percentage character. For example: `Select * From TableName Where ColumnName LIKE '@%TEXT%'`.
:::

:::note
To use the word `variable` within a query, use the following convention: `@%variable%` (else it is referred to as a system variable).
:::
