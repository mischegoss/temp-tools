## Activity Description

Creates a new IBM Tivoli Omnibus alert.

## Output

The alert's Serial ID.

## Settings


* **Module Name** – The name of the IBMTO Module in VAR::PRODUCT_FULL Integraton Module.
* **Properties** – The properties to add to the new record or remove from it. You can add desired fields in the Property Name column, and their associated values in the Property Value column.
