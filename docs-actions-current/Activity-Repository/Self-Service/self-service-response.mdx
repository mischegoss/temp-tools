## Activity Description

Sends a response to the Self Service request. The response can be sent to the user who launched the Self Service form, or to another workflow that ran in order to populate a list in the Self Service form. This can be plain text, a variable, a table, or combination.

## Output

Success/Failure

## Settings


* **Response** — The response to send to the Self Service request. Note that this field can contain either text or a result set table, but not both at the same time.  
  Example: `%DiskSpace% available on %Disk%`
