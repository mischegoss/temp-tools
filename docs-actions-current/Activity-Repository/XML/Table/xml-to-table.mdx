## Activity Description

Converts a specific table within the XML to a table.

## Output

The table.

## Settings


* **XML Session** – The XML session.
* **Table Name** – The name of the element that represents the table.  
   Example: `Files`
* **Property Name** – The name of a single column within the table to convert.
