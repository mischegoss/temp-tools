## Activity Description

Converts a Table to an XML.

## Output

XML

## Settings


* **Remove XML Header** – Creates the XML without a header.
* **Table Variable** – The variable which holds the table to convert.  
   Example: `%readFile1%`
* **Table Name** – The name of the table within the XML that will contain the table information.
