## Activity Description

Removes the selected attribute from the specified Xpath.

## Output

Success/Failure.

## Settings


* **XML Session** – The XML session.
* **XPath Query** – The path of the node from which to remove the attribute.  
   Example: `//configuration/Testing`
* **Attribute Name** – The name of the attribute to remove.
