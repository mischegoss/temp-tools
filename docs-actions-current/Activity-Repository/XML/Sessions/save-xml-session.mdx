## Activity Description

Saves the XML session to a file.

## Output

Success/Failure

## Settings


* **XML Session** – The XML session to save.
* **Path** – The location in which to save the XML file.  
   Example: `C:\conf.xml`
* **Host Name** – The host name or IP address.
* **User Name** – If required, the user name of an account that has read-write access to the host.
* **Password** – If required, the password of the account listed in the User Name field.
* **Append File** – Determines whether to override an existing file or append new content to it.
