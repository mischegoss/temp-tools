## Activity Description

Sends a Syslog message (creates a log entry).

## Settings


* **Host Name** – The Syslog daemon's host name (e.g., SyslogServer).
* **Facility** – The facility category to which the message belongs (e.g., Kernel).
* **Level** – The message's severity level (e.g., Error).
* **Message** – The log entry's text.
* **Port** – The port to use to send the message.
