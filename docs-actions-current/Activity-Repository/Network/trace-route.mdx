## Activity Description

Returns the route over the network between two systems, listing all the intermediate routers a connection must pass through to get to its destination.

## Output

The route.

## Settings


* **Address** – The end point IP address of the route to be traced.
* **Max Hops** – The maximum hops in the returned route value.
* **Hop Timeout** – The number of seconds allocated for one route hop tracing.
