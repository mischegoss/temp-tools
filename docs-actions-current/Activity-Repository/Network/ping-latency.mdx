## Activity Description

Retrieves the elapsed time passing between sending a ping command and retrieving an answer.

## Output

A table with the RoundTrip time and Time to live, or a single value.

## Settings


* **HostName** – The address to ping.
* **Buffer Size** – The ping packet size (the size range is 1 to 65500).
* **Output** – RoundTrip time/Time to live/Both
