## Activity Description

Returns a list of TrueSight events that match the specified criteria.

## Output

A ResultSet of the matching events.

## Settings


* **Module Name** – the module in which to search for matching events.
* **Output Format** –the format of the form.
* **Output Type** – the output type of the form.
* **Filters** – the rules against which to compare the events (add/edit/remove rules from the filter list).
