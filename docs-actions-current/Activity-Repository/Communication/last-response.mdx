## Activity Description

Contacts the latest recipient who took responsibility for the Workflow, via any former communication Activity.

## Settings


* **Message Type –** The type of message sent to the recipient:  
   * **Original** – The message that triggered the Workflow.  
   * **Custom** – Any entered free text.  
   * **Template** – A predefined message.    
   :::note
   Message templates are created via the General page of the Repository section.
   :::
* **Template Name** – Sends a predefined message template
* **Send Recovery Notification –** Sends a notification to the recipient.
