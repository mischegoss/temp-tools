## Activity Description

Sends an email via SMTP.

## Settings


* Message:  
   * **To** – An email address.  
   :::note
   You may add multiple email addresses in the following convention: `<address1>;<address2>`.
   :::
   * **CC** – The email's CC recipient/s.  
   * **BCC** – The email's BCC recipient/s.  
   * **Subject** – The email's subject.  
   * **Body -** The text of the email (you may edit the text style as you wish).
* **Server:** Send email to a specific user using specific SMTP settings.
   * **Name** – The display name of the sender.  
   * **Email address** – The email address of the sender.  
   * **SMTP Server** – The server through which the email is sent.  
   * **Port** – The SMTP port
* **Server Connection:**
   * **Use encrypted connection** – Type of encryption, SSL or TSL  
   * **Requires authentication** – Configure the username and password  
   * **Require secure password authentication** – Check to use SPA
