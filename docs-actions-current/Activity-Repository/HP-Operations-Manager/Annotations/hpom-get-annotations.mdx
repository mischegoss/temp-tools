## Activity Description

Retrieves a list of all annotations of a specified alert in HPOM.

## Output

A ResultSet of all annotations

## Settings


* **Module name** – The name of the selected HPOM Module.
* **Alert Id** – The alert of the annotations in VAR::PRODUCT_FULL to retrieve (e.g., `%hpomCreateAlert1%`).
