## Activity Description

Removes an annotation from an existing open alert in HPOM.

## Output

Success/Failure

## Settings


* **Module Name** – The name of the selected HPOM Module.
* **Alert Id** – The alert in VAR::PRODUCT_FULL to which the new annotation is added (e.g., `%hpomCreateAlert1%`).
* **Annotation Id** – The annotation in Actions to remove (e.g., `%hpomAddAnnotation1%`).
