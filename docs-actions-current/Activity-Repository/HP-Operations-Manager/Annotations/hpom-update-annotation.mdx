## Activity Description

Updates an existing annotation of a specific alert in HPOM.

## Output

Success/Failure

## Settings


* **Module Name** – The name of the selected HPOM Module.
* **Alert Id** – The alert in VAR::PRODUCT_FULL to which the annotation is assigned (e.g., `%hpomCreateAlert1%`).
* **Annotation Id** – The annotation in VAR::PRODUCT to update (e.g., `%hpomCreateAnnotation1%`).
* **Message** – The update text.
