## Activity Description

Adds an annotation to an existing open alert in HPOM.

## Output

The ID of the new annotation

## Settings


* **Module Name** – The name of the selected HPOM Module.
* **Alert Id** – The alert in VAR::PRODUCT_FULL to which the new annotation is added (e.g., `%hpomCreateAlert1%`).
* **Message** – The annotation to add.
