## Activity Description

Creates or sets a custom attribute to an existing alert in HPOM.

## Output

Success/Failure

## Settings


* **Module Name** – The name of the selected HPOM Module.
* **Alert Id** – The name of the alert in VAR::PRODUCT_FULL (e.g., `%hpomCreateAlert1%`).
* **Attribute Name** – The name of the new/existing attribute.  
   Example: `Domain`
* **Attribute Value -** The value to set to the new/existing attribute.
