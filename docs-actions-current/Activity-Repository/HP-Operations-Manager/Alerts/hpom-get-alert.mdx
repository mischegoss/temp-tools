## Activity Description

Retrieves a list of alerts according to the selected criteria.

## Output

A ResultSet of all matching alerts.

## Settings


* **Module Name** – The name of the selected HPOM Module.
* **Filters** – The list of properties used to filter the alerts.  
   * **Name** – The name of the new property (e.g., Severity).  
   * **Relation** – The operator of rule creation (e.g., Equals).  
   * **Value** – The value against which the property is examined (e.g., Minor).
