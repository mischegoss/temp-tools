## Activity Description

Acknowledges an existing open alert in HPOM.

## Output

Success/Failure

:::note
Before the alert is acknowledged, it is owned by the user set in the module's configuration.
:::

## Settings


* **Alert Id:** The alert in Actions to acknowledge (e.g., `%hpomCreateAlert1%`).
