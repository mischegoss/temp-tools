## Activity Description

Updates a record in a Remedyforce form.

## Output

Success/Failure.

## Settings


* **Module Name** – the module in which to update the record.
* **Form Name** – the form in which to update the record.
* **Record ID** – the form's record ID.
* **Properties** – the properties to add to the record or remove from it.
