## Activity Description

Modifies an attribute of an MS Exchange mailbox.

:::note
The activity does not support a 32-bit MS Windows installation. If the VAR::PRODUCT_FULL server is installed on a 32-bit server, contact [Resolve Support](https://support.resolve.com) for additional information on how to use the activity.
:::

## Output

Success/Failure.

:::note
A "Success" output might be returned although the actual PowerShell operation returned a warning.
:::

## Settings


* **Account Name –** The user's logon name (e.g., `John`).
* **Property Name** – The name of the attribute to modify.
* **Value** – The selected attribute's new value.  
  Example: `%newalias%`
* **Host Name** – The Exchange server's name.
* **User Name** – If required, the user name of account that has read-write access to the host.
* **Password** – If required, the password of the account listed in the User Name field.

:::note
The following table details the optional properties and the data units they are assigned:

| Property                 | Data Unit |
|--------------------------|-----------|
| IssueWarningQuota        | MB        |
| ProhibitSendQuota        | MB        |
| ProhibitSendReceiveQuota | MB        |
| Max Receive              | KB        |
| Max Send                 | KB        |
| All other                | Bytes     |
