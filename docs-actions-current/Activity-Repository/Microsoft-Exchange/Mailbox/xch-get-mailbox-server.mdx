## Activity Description

Retrieves a mailbox's server objects from an MS Exchange server.

:::note
The activity does not support a 32-bit MS Windows installation. If the VAR::PRODUCT_FULL server is installed on a 32-bit server, contact [Resolve Support](https://support.resolve.com) for additional information on how to use the activity.
:::

## Output

The selected server object.

## Settings


* **Host Name** – The Exchange server's name.
* **User Name** – If required, the user name of an account that has read-write access to the host.
* **Password** – If required, the password of the account listed in the User Name field.
* **Property Name** – The name of the selected property.
