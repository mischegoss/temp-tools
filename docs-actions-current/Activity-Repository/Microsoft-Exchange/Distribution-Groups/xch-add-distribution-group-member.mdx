## Activity Description

Adds a user to an MS Exchange distribution group.

:::note
The activity does not support a 32-bit MS Windows installation. If the VAR::PRODUCT_FULL server is installed on a 32-bit server, contact [Resolve Support](https://support.resolve.com) for additional information for additional information on how to use the activity.
:::

## Output

Success/Failure

## Settings


* **Account Name** – The user's logon name (e.g., `John`).
* **Group Name** – The distribution group in which to add the user (e.g., `IT_Admin`).
* **Host Name** – The Exchange server's name.
* **User Name** – If required, the user name of an account that has read-write access to the host.
* **Password** – If required, the password of the account listed in the User Name field.
