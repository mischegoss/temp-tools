title: Jira Overview
---

Jira activities are used to manage Jira tickets and can be customized according to your Jira project's requirements.

:::note
After dragging each activity and dropping it in the Sequential Workflow pane, you will be asked to define two parameters: issue type and project name. Once you have selected the issue and project, filled in the information in the next window and clicked OK, you will not be able to change these parameters.
:::

:::note
The activities are displayed as they appear by default, without any user customization.
:::

:::note
The integration module user must have Jira Administrator permissions to configure and manage Jira projects in Actions.
:::
