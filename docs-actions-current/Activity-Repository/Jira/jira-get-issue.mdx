## Activity Description

Gets a list of Jira issues according to the selected criteria.

## Output

A ResultSet of all matching issues.

## Settings


* **Module Name** – The Jira module.
* **Project Name** – The Jira project in which to create the new issue.
* **Output Format** – Select the output format of how you’d like to see the values displayed.
* **Output Type** – Select the output type you’d like to see the values displayed in.
* **Filters** – Filters are applied (by the activity) to issues which contain values in the specified field. Click **Add** to add filters to the list, or **Edit** to modify the existing filters.
