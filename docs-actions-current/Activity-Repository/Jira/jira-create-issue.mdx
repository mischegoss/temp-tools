## Activity Description

Creates a new Jira issue.

## Output

The ID of the new issue.

## Settings


* **Module Name** – The Jira module.
* **Project Name** – The Jira project in which to create the new issue.
* **Issue Type** – The type of the new issue.
* **Properties** – The properties of the selected issue type.


:::note
The **Create Issue** activity allows setting only one value to Label, Multiple Choice, Radio Button and Checkbox type properties. To add more values or remove them, use the Update Issue activity.
:::
