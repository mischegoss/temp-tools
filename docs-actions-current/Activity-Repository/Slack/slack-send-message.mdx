## Activity Description

Sends a Slack message to a user or channel using the API endpoing [chat.postMessage](https://api.slack.com/methods/chat.postMessage).

## Output

Success/Failure

## Settings

* **Slack Bot Token** – Uses token from Global variables.
* **Channel or user**: add the user ID or the channel name or ID
**Note**: You may use the channel name, but you cannot use a user name.
* **Text to send**: add the text to send in your Slack message
