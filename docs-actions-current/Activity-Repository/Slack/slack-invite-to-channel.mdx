## Activity Description

Invites the defined user to the selected channel using the API endoint [conversations.invite](https://api.slack.com/methods/conversations.invite).

## Output

Success/Failure

## Settings


* **Slack Bot Token** – Uses token from Global variables.
* **Channel ID** – The ID of the channel to archive.
* **User IDs (comma separated)** - The user IDs of users to be invited to the channel.
