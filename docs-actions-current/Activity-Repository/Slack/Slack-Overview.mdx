Slack is a collaboration tool and service. Slack channels provide a single place for messaging, tools, and files.

To learn more about integrating Slack activities in workflows, see [Configuring the Slack Integration](../../Developing-Custom-Activities/Activity-Guides/Slack-Integration/Configuring-the-Slack-Integration.mdx) and [Adding Slack Activities](../../Developing-Custom-Activities/Activity-Guides/Slack-Integration/Adding-Slack-Activities.mdx).

The available activities are: 
|Activity|Description|
|-------|---------|
|[Archive Channel](./slack-archive-channel.mdx)|Archives a Slack channel when defined workflow conditions are met.|
|[Create Channel](./slack-create-channel.mdx)| Creates  a new Slack channel as part of a workflow.|
|[Invite to Channel](./slack-invite-to-channel.mdx)| Adds a user to a Slack channel.|
|[Kick from Channel](./slack-kick-from-channel.mdx)| Removes a user from a Slack channel.|
|[List Users](./slack-list-users.mdx)| Generates a list of users in a channel.|
|[Send Message](./slack-send-message.mdx)| Sends a Slack message with content defined in the workflow.|

:::note
Currently, you can only use the Slack activities provided out of the box. The ability to create and manage your own custom Slack endpoints will be available in a future release.
:::
