## Activity Description

Archives a Slack channel using the API endpoint [conversations.archive](https://api.slack.com/methods/conversations.archive).

## Output

Success/Failure

## Settings


* **Slack Bot Token** – Uses token from Global variables.
* **Channel ID** – The ID of the channel to archive.
