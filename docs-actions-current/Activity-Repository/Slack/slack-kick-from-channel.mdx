## Activity Description

Kicks the defined user out of the selected channel using the API endpoint [conversations.kick](https://api.slack.com/methods/conversations.kick).

## Output

Success/Failure

## Settings


* **Slack Bot Token** – Uses token from Global variables.
* **Channel ID** – The ID of the channel to archive.
* **User ID** - The user ID of users to be invited to the channel.
