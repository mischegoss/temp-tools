## Activity Description

Creates a Slack channel using the API endpoint [conversations.create](https://api.slack.com/methods/conversations.create).

## Output

Success/Failure

## Settings


* **Slack Bot Token** – Uses token from Global variables.
* **Channel Name** – The name of the channel to archive.
