## Activity Description

Returns a table that contains the last scan date, the number of scanners, and the number of scanners that found this hash malicious.

## Output

The last scan date, the amount of engines that scanned the hash, and the amount of engines discovering the hash to be malicious.

## Settings


* **Hash** – File hash of the file for inspection.
* **Key** – Authentication key for VirusTotal API key for access.
* **Proxy** The internal proxy to go through to reach the external VirusTotal system.
* **User Name/Password** – The credentials for the proxy.
