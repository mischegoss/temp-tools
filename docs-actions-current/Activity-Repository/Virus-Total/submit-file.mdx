## Activity Description

Returns a table that contains the last scan date, the number of scanners, and the number of scanners that found this file malicious.

## Output

The last scan date, the amount of engines that scanned the file, and the amount of engines discovering the file to be malicious.

## Settings


* **Path** – The path to file local to the Executor for inspection.
* **Key** – Authentication key for VirusTotal API key for access.
* **Proxy** The internal proxy to go through to reach the external VirusTotal system.
* **User Name/Password** – The credentials for the proxy.
