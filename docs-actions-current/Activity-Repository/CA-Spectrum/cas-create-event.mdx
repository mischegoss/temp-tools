## Activity Description

Creates a new event in CA Spectrum.

:::note
This is an event and not an alarm.
:::

## Output

Success/Failure

## Settings
* **Module Name**—The CA Spectrum module name.
* **Model Name**—The CA Spectrum model name.
* **Event Code**—The ID code of the new CA Spectrum event.
* **Properties**—Click **Property Id/Property Value** to add or remove properties to the selected event.

:::note
The CA Spectrum API supports only string type properties.
:::
