## Activity Description

Allocates a value to one or more Workflow/Global variables.

## Settings

* **Memory Scope (Workflow/Global)** - Workflow: The variable is accessed only by this workflow. Global: The variable is accessed by all workflows.
* **Name** - The variable's name to be referenced (the name must differ from the activity's name). The name must start with a letter and contain only the A-Z, a-z, 0-9 characters.
* **Value** - The variable's initial value - a constant value or another variable (variable names must follow the convention `%Variable%`).

:::note Copy Excel Table Data
You can paste data copied from Microsoft Excel to fill in the table in this activity.
:::
