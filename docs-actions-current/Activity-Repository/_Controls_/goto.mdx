## Activity Description

Redirects to another activity in the workflow (according to its name) and executes it. This activity is commonly used in cases of any failure (e.g., phone call establishment failure) to repeat the activity that failed to execute.

## Settings


* **Go to Activity** — The name of the activity instance in the workflow that the Goto activity redirects to.  
  Example: `telnet1`
