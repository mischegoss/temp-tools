## Activity Description

Returns the number of currently active instances of the selected Workflow.

## Output

The number of instances.

## Settings


* **Use current Workflow/Select a different Workflow** - Determines whether to count the instances of the current Workflow or another one.
* **Workflow Name** - In case "Select a different Workflow" is selected in the previous step, this parameter sets the Workflow of which to count the instances.
