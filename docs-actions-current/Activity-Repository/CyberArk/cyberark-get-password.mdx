## Activity Description

This activity retrieves a password from a CyberArk safe.

:::note
To use this activity, verify that the CyberArk SDK .dll file "NetPasswordSDK.dll" exists in the VAR::PRODUCT_FULL Server installation folder. This .dll file will not be provided by Resolve.
:::

## Settings


* **Safe** – The name of the CyberArk safe.  
   Example: `AcmeSafe`
* **Folder:** The name of the folder in which the safe is located.  
   Example: `root`
* **Application:** The name of the application for which to retrieve a password.  
   Example: `Resolve Actions`
* **Object:** The name of the object for which to retrieve a password.  
   Example: `OperatingSystem-WinDomain-127.0.0.1`
* **Port:** The port through which to connect to the CyberArk safe (the default port is 18923).
* **Reason**: A free text field used for adding comments.  
   Example: `Retrieve a password - Resolve Actions`
