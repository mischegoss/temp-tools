## Activity Description

Updates a case in HP ArcSight.

## Output

Success/Failure

## Settings


* **Module Name** – The HP ArcSight module with which the activity is associated.
* **Case Id** – The ID of the case to update.  
   Example: `%CaseID%`
* **Properties** – Optional properties to add fields to the list, edit or remove them.
