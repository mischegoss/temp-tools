## Activity Description

Gets a list of HP ArcSight events according to the specific case ID and the selected time frame and criteria.

## Output

A ResultSet of all matching events.

## Settings


* **Module Name** – The HP ArcSight module name.
* **Case Id** – The ID of the case.  
   Example: `%CaseID%`
* **Filters** – Add new filters to the list, or edit existing ones. Filters are applied (by the activity) to cases which contain values in the specified field.
