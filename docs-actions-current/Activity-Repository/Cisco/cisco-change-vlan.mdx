## Activity Description

Changes the Cisco Vlan.

## Output

Success/Failure.

## Settings


* **Cisco session** – the session on which to send the command.
* **Password** – the related password.
* **Switch Port Mode** – the mode of the switch port: Access or Trunk
* **Interface name** – the interface name.  
   Example: `GigabitEthernet/1/0/3`
* **New Vlan** – the name of the new Vlan.  
   Example: `GBEth/2/0/3`
* **Trunk Vlans** – the name of the Trunk Vlans.
