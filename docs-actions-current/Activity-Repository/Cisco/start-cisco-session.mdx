## Activity Description

Starts a Cisco router session.

## Output

Success/Failure.

## Settings


* **Use Secure Connection** – determines whether to use a Telnet/SSH connection.
* **Session maximum time** – the maximum length of the session: `HH:MM:SS`.  
   Example: `00:10:00`
