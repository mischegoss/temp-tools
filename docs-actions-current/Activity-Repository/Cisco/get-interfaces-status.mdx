## Activity Description

Retrieves a ResultSet of the Cisco session's related interfaces and their status (up/down).

## Output

A list of interfaces and their statuses.

## Settings


* **Cisco Session** – the session from which to derive the list of interfaces.
* **Interface name** – the interface name for which to search.  
   Example: `GigabitEthernet/1/0/3`
