## Activity Description

Changes a user/computer account's attribute, including creating custom properties.

## Output

Success/Failure

## Settings


* **Account Type** – The account type (e.g., user/computer).
* **Name** – The account name (e.g., `John`).
* **Host Name** – The name or IP address of the Active Directory's server.
* **User Name** – If required, the user name of an account with read-write access to the Active Directory.
* **Password** – If required, the password of the account listed in the User Name field.
* **Property Name** - The name of the property. To create a unique property, type in a name.
* **Property Value** – The new value applied to an existing property (e.g., `john_smith@example.com`).
* **Id Name** - For custom properties, the ID to retrieve data in workflows.
