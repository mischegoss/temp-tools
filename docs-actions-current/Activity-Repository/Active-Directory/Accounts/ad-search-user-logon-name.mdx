## Activity Description

Searches for users’ logon names according to predefined criteria.

## Output

A list of logon names which match the search criteria.

## Settings


* **Host Name** – The name or IP address of the Active Directory's server.
* **User Name** – If required, the user name of an account that has read-write access to the Active Directory.
* **Password** – If required, the password of the account listed in the User Name field.
* **Property Name/Property Value** – the criteria according to which the search is performed, e.g., first name, last name, email (e.g., in first name, use `%FirstName%` or `John`). To represent wildcards, use `*`.
