:::note
Active Directory Activities allow you to manage your Active Directory accounts (e.g., creating user accounts, deleting user accounts, changing the attributes of a user account or a group).
:::

In order to execute Active Directory activities, the following ports must be open between Resolve Actions and the Active Directory server:

* Port 389 – LDAP
* Port 445 – NTLM
