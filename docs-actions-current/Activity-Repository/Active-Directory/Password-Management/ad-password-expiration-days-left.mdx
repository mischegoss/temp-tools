## Activity Description

Retrieves the number of days left until a user's password expires.

## Settings


* **Account Name** – The account name (e.g., `John`).
* **Host Name** – The name or IP address of the Active Directory's server.
* **User Name** – If required, the user name of an account that has read-write access to the Active Directory.
* **Password** – If required, the password of the account listed in the User Name field.
* **Use Active Directory password max age** – use the number defined by the "Active Directory password".
* **Manual define age** – manually define the number of days before the user's password expires.
* **Age** – When choosing **Manual define age**, the number of days that should be used to set password expiration.
