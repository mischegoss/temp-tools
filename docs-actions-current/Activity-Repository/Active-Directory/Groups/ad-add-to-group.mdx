## Activity Description

Adds a user or a device to a group in the Active Directory.

## Output

Success/Failure

## Settings


* **Account Type** – Determines whether the account is a user account or a computer account.
* **Name** – The user's logon name (e.g., "John") or the computer name.
* **Host Name** – The name or IP address of the Active Directory's server.
* **User Name** – If required, the user name of an account that has read-write access to the Active Directory.
* **Password** – If required, the password of the account listed in the User Name field.
* **Group Name** – the group to which the user or computer is added (e.g., `Resolve_IT`).
