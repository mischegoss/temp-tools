## Activity Description

Creates a new record in BMC Helix Remedyforce based on defined properties.

## Output

The record ID.

## Settings


* **Module Name** – the module in which to create the new record.
* **Form Name** – the form in which to create the new record.
* **Properties** – the properties to add to the new record.
