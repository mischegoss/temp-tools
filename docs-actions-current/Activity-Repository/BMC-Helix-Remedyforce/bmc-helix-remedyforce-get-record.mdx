## Activity Description

Returns a list of BMC Helix Remedyforce records that match the specified criteria.

## Output

A ResultSet of the matching records.

## Settings


* **Module Name** – the module in which to search for matching records.
* **Form Name** – the form in which to search for matching records.
* **Requested Fields** – select the desired fields for searching.
* **Output Format** –the format of the form.
* **Output Type** – the output type of the form.
* **Filters** – the rules against which to compare the records (add/edit/remove rules from the filter list).
