## Activity Description

Sums up a sequence of table cell values. Text characters are ignored.

## Output

The sum value.

## Settings


* **Table Variable** – The name of the table (variable names must follow the convention %Variable%).
* **Rows/Columns (From)** – The row/column index of the first cell that indicates the sequence's beginning.
* **Rows/Columns (To)** – The row/column index of the last cell that indicates the sequence's end.
