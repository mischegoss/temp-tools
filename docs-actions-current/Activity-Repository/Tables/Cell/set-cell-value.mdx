## Activity Description

Sets a new value to a cell in a table.

## Settings


* **Table Variable** – The name of the table (variable names must follow the convention %Variable%).
* **Row Number** – The cell's row number.
* **Column Type (Name/Number)** – The cell's data type.
* **New Value** – The new value set to the cell.
