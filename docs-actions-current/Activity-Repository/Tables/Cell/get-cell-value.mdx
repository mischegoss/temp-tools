## Activity Description

Retrieves a cell's value from a table.

## Settings


* **Table Variable** – The name of the table (variable names must follow the convention %Variable%).
* **Row Number** – The cell's row number.
* **Column Type** – The cell's data type.
* **Column Number/Name** – The cell's column number/name.
