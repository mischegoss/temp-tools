## Activity Description

Adds a new row (or several rows) to a table.

## Settings


* **Table Variable** – The name of the table (variable names must follow the convention `%Variable%`).
* **Number of Rows** – The number of rows added to the table.
* **At the Beginning of the Table/At the End of the Table/After Row Number** – The location where the new row/s is/are created.
