## Activity Description

Returns a new table with updated values based on key pair mappings defined in a table of global variables.

## Settings


* **Source Table** – The name of the source table (e.g., %ReadXLS1%).
* **Mapping Table** – The name of the mapping table (e.g., %NewValuesTable%).
