## Activity Description

Compares two tables (or two specific columns in the tables), and displays the equivalent/nonequivalent rows.

## Settings


* **Original Table Variable** – The name of the first table (e.g., %ReadXLS1%).
* **Altered Table Variable** – The name of the second table (e.g., %ReadXLS2%).
* **Specific Column** – Check to enter a specific location of the column to compare (optional).
* **Column** – The location of the column to compare.
* **Show Equal Rows** – Determines whether to display equivalent/nonequivalent rows.
