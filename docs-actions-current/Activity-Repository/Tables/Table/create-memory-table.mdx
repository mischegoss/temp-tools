## Activity Description

Creates a ResultSet table and saves it to the memory.

## Settings

* **Table Name** – The name of the new table (e.g., %ResultSet%).
* **Columns** – The number of columns created in the new table.
* **Rows** – The number of rows created in the new table.

:::note Copy Excel Table Data
You can paste data copied from Microsoft Excel to fill in the table in this activity.
:::
