## Activity Description

Calculates the placement rating for one or more hosts managed by SCVMM on which you might want to deploy a specific virtual machine.

## Output

The host's rating.

## Settings


* **Host Name** – The host name of the virtual machine.
* **User Name** – If required, the username of an account that has read-write access to the HyperV.
* **Password** – If required, the password of the account listed in the User Name field.
* **Template name** – The name of the template.
* **Cluster name** – If required, the name of the cluster.
* **Disk space** – The space of the disk.  
   Example: `1Gb`
