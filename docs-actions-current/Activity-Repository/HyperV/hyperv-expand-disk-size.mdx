## Activity Description

Expands the disk size on the virtual server.

## Output

Success/Failure

## Settings


* **Host Name** – The host name of the virtual machine.
* **User Name** – If required, the username of an account that has read-write access to the HyperV.
* **Password** – If required, the password of the account listed in the User Name field.
* **VM Name** – The name of the VM.
* **Size** – The size of the disk expansion.  
   Example: `10Mb`
* **Bus** – The bus of the disk expansion.
* **Lun** – The LUN of the disk expansion.
