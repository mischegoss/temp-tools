## Activity Description

Adds a disk to a virtual machine.

## Output

Success/Failure

## Settings


* **Host Name** – The host name of the virtual machine.
* **User Name** – If required, the username of an account that has read-write access to the HyperV.
* **Password** – If required, the password of the account listed in the User Name field.
* **VM Name** – The name of the VM.
* **File name** – The name of the file to add.  
   Example: `Disk5.vhdx`
* **Size** – The size of the additional disk to add.  
   Example: `10Mb`
* **Type** – The type of the additional disk to add.
* **Bus** – The bus of the additional disk to add.
* **Lun** – The LUN of the additional disk to add.
