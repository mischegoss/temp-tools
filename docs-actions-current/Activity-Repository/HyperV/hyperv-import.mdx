## Activity Description

Imports a virtual server.

## Output

Success/Failure

## Settings


* **Host Name** – The host name of the virtual machine.
* **User Name** – If required, the username of an account that has read-write access to the HyperV.
* **Password** – If required, the password of the account listed in the User Name field.
* **Template name** – The name of the template.
* **Template path** – The path of the template.  
   Example: `C:\hypervTemplates\Virtual Machine.vhdx`
* **Destination name** – The name of the destination to import.  
   Example: `VM1`
* **Destination path** – The path of the destination to import.  
   Example: `C:\hyoervTemplates`
* **Network adapter** – The network adapter.  
   Example: `Internal1`
