title: HyperV Overview
---

HyperV activities help you to manage your HyperV virtual environment.

:::note
Some activities support setting only the HyperV server or only the SCVMM server, according to the specific action.
:::

:::note
Set the SCVMM checkbox to force the activity to approach the SCVMM server only.
:::
