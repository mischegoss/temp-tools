## Activity Description

Creates a new virtual server.

## Output

Success/Failure

## Settings


* **Host Name** – The host name of the virtual machine.
* **User Name** – If required, the username of an account that has read-write access to the HyperV.
* **Password** – If required, the password of the account listed in the User Name field.
* **VM Name** – The name of the VM.
* **VHD Path** – The VHD path.  
   Example: `C:\ProgramData\Microsoft\Windows\Hyper-V\VMS\Blank Disk - Small.vhd`
* **Template VHD path** – The template VHD path.  
   Example: `C:\Temp\Blank Disk - Small.vhd`
* **Disk Space (MB)** – The disk space of the new virtual server.
* **Memory Size (MB)** – The memory size of the new virtual server.
* **Switch Name** – The name of the switch.  
    Example: `V-Switch`
* **Number of Processors** – The number of the processors.
