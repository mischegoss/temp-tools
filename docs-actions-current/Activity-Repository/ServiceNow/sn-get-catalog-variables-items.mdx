## Activity Description

Gets all catalog variables associated with the selected record.

:::note
This activity only works for records of the type Request Item or Task.
:::

## Output

The ID of the new record.

## Settings


* **Module Name** – The name of the ServiceNow module in VAR::PRODUCT_FULL.
* **From Name** – Choose to query variables from a task or request item.
* **Record Number** – Enter the task or request item number to retrieve variables from.  
   Example: `snGetRecord1`
