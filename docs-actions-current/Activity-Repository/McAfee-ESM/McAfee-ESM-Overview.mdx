McAfee ESM activities are used in the VAR::PRODUCT_FULL McAfee ESM integration module that provides a bidirectional communication channel between McAfee ESM and VAR::PRODUCT. After you add and configure the module, AR::PRODUCT pulls new created alerts and updates, translates them into incidents, and displays them in the Actions dashboard.

:::note
To use these activities, you need to have a configured and enabled McAfee ESM integration module.
:::
