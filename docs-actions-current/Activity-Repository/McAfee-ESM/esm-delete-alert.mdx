## Activity Description

Deletes an existing alert from the McAfee ESM console.

:::note
This activity doesn't verify the validity or existence of the provided ID, as the McAfee ESM API doesn't allow it.
:::

## Output

Success/Failure

## Settings
* **Module Name**—The McAfee ESM module name.
* **Alert ID**—The McAfee ESM alert ID.
