## Activity Description

Retrieves a list of Cherwell records based on the selected criteria.

## Output

Success/Failure

## Settings


* **Module Name** – The name of the Cherwell module.
* **Form Name** – The name of the Cherwell form.
* **Output Format** – The output format of the form.
* **Output Type** – The output type of the form.
* **Properties** - The properties to add to the Properties section or remove from it.  
   Field names are values to be used to create the new record.
