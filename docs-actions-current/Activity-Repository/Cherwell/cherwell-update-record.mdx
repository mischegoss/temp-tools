## Activity Description

Updates a record in Cherwell.

## Output

Success/Failure.

## Settings


* **Module Name** – The name of the Cherwell module.
* **Form Name** – The name of the Cherwell form.
* **Properties** – The properties to add to the Properties section or remove from it.  
   Field names are values to be used to update the record (**Number** is a mandatory property).
