## Activity Description

Retrieves a list of attachments according to the selected Cherwell form.

## Output

Attachment list.

## Settings

* **Module Name** – The name of the Cherwell module.
* **Form Name** – The name of the Cherwell form.
* **Id Type** – Record or Public Id.
* **Id** – The record Id.  
   Example: `%RecordID%`
* **Attachment Type** – The type of attachment.
