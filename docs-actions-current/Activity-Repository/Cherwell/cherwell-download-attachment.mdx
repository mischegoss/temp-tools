## Activity Description

Downloads an attachment to an existing Cherwell record.

## Output

Downloaded attachment.

## Settings


* **Module Name** – The name of the Cherwell module.
* **Form Name** – The name of the Cherwell form.
* **Id Type** – Record or Public Id.
* **Id** – Id of the record.  
   Example: `%RecID%`
* **Attachment Id** – Id of the attachment.  
   Example: `%AttachmentID%`
* **Name** – The attachment filename.
* **Path** – Attachment path.
