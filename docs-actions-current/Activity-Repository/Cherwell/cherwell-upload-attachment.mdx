## Activity Description

Uploads an attachment to an existing Cherwell record.

## Output

Uploaded attachment.

## Settings

* **Module Name** – The name of the Cherwell module.
* **Form Name** – The name of the Cherwell form.
* **Id Type** – Record or Public Id.
* **Id** – The record Id.  
   Example: `%RecID%`
* **Path** – The path of the attached file to upload.
