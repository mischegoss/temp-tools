## Activity Description

Adds a journal note to a specific Cherwell record.

## Output

Note added/failure.

## Settings


* **Module Name** – The name of the Cherwell module.
* **Form Name** – The name of the Cherwell form.
* **Id Type** – Record of Public Id.
* **Id** – Id of the journal note.  
   Example: `103729`
* **Properties** - The properties to add to the Properties section or remove from it.
