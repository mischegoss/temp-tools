## Activity Description

Gets a list of active SolarWinds alerts according to the selected criteria.

## Output

A result set or json message of all matching alerts.

## Settings


* **Module Name** – The name of the SolarWinds Module in VAR::PRODUCT_FULL.
* **Output Format** – Select depending on how you'd like to see the value displayed.
* **Output Type** – Select depending on how you'd like to see the value displayed.
* **Filters** – Applied (by the activity) to alerts which contain values in the specified field. Since retrieving the translation of fields’ data from SolarWinds might take a while, it is recommended that the filters of this activity are as specific as possible.
