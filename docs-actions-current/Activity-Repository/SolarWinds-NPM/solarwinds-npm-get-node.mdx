## Activity Description

Retrieves a list of SolarWinds NPM nodes according to the selected criteria.

## Output

A result set of all matching nodes.

## Settings


* **Module Name** – The name of the SolarWinds Module in VAR::PRODUCT_FULL.
* **Filters** – Applied (by the activity) to nodes which contain values in the specified field. Since retrieving the translation of fields’ data from SolarWinds might take a while, it is recommended that the filters of this activity are as specific as possible.
