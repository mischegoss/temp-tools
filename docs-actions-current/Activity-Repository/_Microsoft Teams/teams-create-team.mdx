## Activity Description

Create a new team in MS Teams.

## Output

Team created/Failure

## Settings


* **Module Name** — Select the Teams module that will execute this activity.
* **Team Name** — The name of the team to be created, 256-character limit.
* **Description** — The description of the team, 1,024-character limit.
* **Privacy** — The privacy settings of the created team.
