## Activity Description

Delete the specified channel in Teams.

## Output

Channel deleted/Failure

## Settings


* **Module Name** — Select the Teams module that will execute this activity.
* **Team Name** — The name of the team under which this channel exists.
* **Channel Name** — The name of the channel to be deleted.

:::note
At this time, only unique team names are supported. Trying to delete a channel under a team where the name is not unique within your Teams organization will fail.
:::
