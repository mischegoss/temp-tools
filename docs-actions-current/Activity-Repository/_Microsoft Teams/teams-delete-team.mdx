## Activity Description

Delete the specified team in Teams.

## Output

Team deleted/Failure

## Settings


* **Module Name** — Select the Teams module that will execute this activity.
* **Team Name** — The name of the team that will be deleted.

:::note
At this time, only unique team names are supported. Trying to delete a team where the name is not unique within your Teams organization will fail.
:::
