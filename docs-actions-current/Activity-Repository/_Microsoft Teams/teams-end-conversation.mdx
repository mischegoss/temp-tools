## Activity Description

Terminates the existing Teams session. No additional messages will be routed to the current workflow. The next incoming message will be a new session

## Output

End session/Failure

## Settings


* **Module Name** — Select the Teams module that will execute this activity.
* **Teams Conversation** — The Conversation ID of the conversation you wish to end.  
   Example: `%conversationId%`
