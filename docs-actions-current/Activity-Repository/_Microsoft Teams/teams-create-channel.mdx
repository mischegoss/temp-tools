## Activity Description

Create a new channel under a team in Teams.

## Output

Channel created/Failure

## Settings


* **Module Name** — Select the Teams module that will execute this activity.
* **Team Name** — The name of the team under which the channel will be created.
* **Channel Name** — The name of the channel to be created.
* **Description** — The description of the channel.

:::note
At this time, only unique team names are supported. Trying to create a channel under a team where the name is not unique within your Teams organization will fail.
:::
