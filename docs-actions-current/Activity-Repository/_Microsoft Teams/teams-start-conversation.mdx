## Activity Description

Start a conversation on MS Teams

## Output

Success/Failure

## Settings


* **Module Name** — Select the Teams module that will execute this activity.
* **Max Conversation Idle Time (Min)** — The amount of time the conversation/session can be idle before it times out and the session ends. The counter is reset after each Teams Send Message activity.
* **Username** — The username of the user who will receive the message.  
   Example: `%User%`
