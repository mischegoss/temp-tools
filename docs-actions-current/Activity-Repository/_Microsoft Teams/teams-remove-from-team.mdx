## Activity Description

Remove the defined user from the selected team.

## Output

User removed from team/Failure

## Settings


* **Module Name** — Select the Teams module that will execute this activity.
* **Team Name** — The name of the team from which users will be removed.
* **Name** — The name of the user to remove from the team.  
   Example: `john@administrator.onmicrosoft.com`

:::note
At this time, only unique team names are supported. Trying to remove a user or from a team where the name is not unique within your Teams organization will fail.
:::
