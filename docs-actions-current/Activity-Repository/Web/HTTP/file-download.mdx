## Activity Description

Downloads a file from a website.

## Output

Success/Failure

:::note
The activity is not a download manager; therefore, network connectivity problems may cause the activity to fail.
:::

## Settings


* **URL** – The address of the website from which to download the file.
* **User Name** – A user authorized to access the website.
* **Password** – The user's password.
* **Proxy** – If the website is accessed via a proxy server, the proxy server's address must be specified.
* **User Name** – The username by which the selected proxy server is accessed (optional, usually not required).
* **Password** – The password by which the selected proxy server is accessed (optional, usually not required).
* **Bypass proxy server for the following addresses** – Used for exceptional addresses not required to pass through the proxy servers.
* **Destination/Path** – The location in which to save the downloaded file.  
   Example: `D:\Installations\`
* **Destination/Host Name** – The destination host name or IP address.
* **Destination/User Name** – If required, the username of an account that has read-write access to the host.
* **Destination/Password** – If required, the password of the account listed in the User Name field.
