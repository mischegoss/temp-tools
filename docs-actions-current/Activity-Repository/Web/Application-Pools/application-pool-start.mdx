## Activity Description

Starts the specified application pool on the designated server.

:::note
Application Pool activities require IIS 6 backward compatibility.
:::

## Output

Success/Failure

## Settings


* **Pool Name–** The started application pool.  
   Example: `%ApplicationPool%`
* **Host Name** – The host name or IP address.
* **User Name** – If required, the user name of an account that has read-write access to the host.
* **Password** – If required, the password of the account listed in the User Name field.
