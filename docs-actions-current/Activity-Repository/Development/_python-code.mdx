## Activity Description

Executes a Python script predefined in Repository > External Development > [Development](../../Product-Navigation/Repository/External-Development/Development.mdx).

## Output

Success/Failure

## Settings
* **Language type**—Non-editable, always set to `Python`.
* **Code name**—The name of the Python script predefined in **Repository > External Development > [Development](../../Product-Navigation/Repository/External-Development/Development.mdx)**.
