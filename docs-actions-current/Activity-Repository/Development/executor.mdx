## Activity Description

Executes an external command or script.

## Settings


* **Command** – The executed command.
* **Arguments** – Overrides the arguments specified in the command created in the Repository.
* **Do not Wait for Result** – Select to avoid waiting for the activity's result before continuing to the next one.
* **Host Name** – The host name or IP address.
* **User Name** – If required, the username of an account that has read-write access to the host.
* **Password** – If required, the password of the account listed in the User Name field.
