## Activity Description

Adds a note to an existing offense.

## Output

Success/Failure

## Settings


* **Module Name** – The name of the IBM QRadar module that will execute this activity.
* **Offense ID** – The ID of the offense the note will be added to.  
   Example: `%offenceID%`
* **Note Text** – The text you want to have for the note.
