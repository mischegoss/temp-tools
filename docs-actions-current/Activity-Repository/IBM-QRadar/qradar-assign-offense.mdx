## Activity Description

Assigns an offense to a specified user.

## Output

Success/Failure

## Settings


* **Module Name** – The name of the IBM QRadar module that will execute this activity.
* **Offense ID** – The ID of the offense you want to assign.  
   Example: `%offenceID%`
* **Username** – The username of the person that the offense will be assigned to.
