## Activity Description

Gets a list of QRadar offenses according to the selected criteria.

## Output

A ResultSet of all the matching offenses.

## Settings


* **Module Name** – The name of the IBM QRadar module that will execute this activity.
* **Filters** – Filters are applied (by the activity) to offenses that contain values in the specified field. Filters should be as specific as possible.
