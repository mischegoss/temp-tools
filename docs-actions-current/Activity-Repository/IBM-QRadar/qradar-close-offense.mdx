## Activity Description

Closes an open offense.

## Output

Success/Failure

## Settings


* **Module Name** – The name of the IBM QRadar module that will execute this activity.
* **Offense ID** – The ID of the offense you want to close.  
   Example: `%offenceID%`
* **Close Reason** – Select the relevant close reason from the list.
