## Activity Description

Returns the amount of messages in a queue.

## Output

The number of messages.

## Settings


* **Type** – The type of message queue (Microsoft Message Queue is currently the only available option).
* **Host Name** – The host name or IP address.
* **Queue Manager** – The queue manager node.
* **Channel** – The channel name.
* **Queue Name** – The queue's name (e.g., "IncomingMSG").
