## Activity Description

Terminates all active instances of a specified process.

## Output

Success/Failure

## Settings


* **Process Type** – The process type can be either Name or Id.
* **Process Name** – The name of the terminated process (e.g., "svchost.exe").
* **Process Id** – The id of the terminated process (if Process Type is Id).
* **Host Name** – The host name or IP address.  
   Example: `%ServerName%`
* **User Name** – If required, the username of an account that has read-write access to the host.
* **Password** – If required, the password of the account listed in the User Name field.
