## Activity Description

Removes characters from the beginning or end of a text.

## Output

The trimmed text.

## Settings


* **Text** – The text to trim.
* **Leading String** – The characters to remove from the beginning of the text.
* **Ending String** – The characters to remove at the end of the text.
