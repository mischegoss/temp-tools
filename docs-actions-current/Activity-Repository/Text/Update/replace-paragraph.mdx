## Activity Description

Replaces one paragraph with other text.

## Output

The new paragraph.

## Settings


* **Source String** – The text in which the replaced paragraph resides.  
   Example: `%body%`
* **From** – The first word of the replaced paragraph.
* **To** – The last word of the replaced paragraph.
* **Replace With** – The new paragraph.  
   Example: `%EmergencyEmail%`
