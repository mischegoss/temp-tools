## Activity Description

Returns the position of the first occurrence of a phrase within a text (or 0 if it does not exist). The search begins at the first character of the text.

## Output

The index of the phrase or 0 if it does not exist.

## Settings


* **Find Text** The phrase to search.
* **Text** – The text in which to search for the phrase.  
   Example: `%logs%`
