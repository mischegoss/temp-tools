## Activity Description

Returns all matches between a Regular Expression and a given text.

## Output

A table of the matching phrases.

## Settings


* **Text to Match** – The text in which the Regular Expression is searched.
* **Regular Expression** – The Regular Expression's syntax.
* **Choose Regular Expression Description** – An email address, an IP address, a URL, or another type.
* **Formula** – The pattern of the Regular Expression.
