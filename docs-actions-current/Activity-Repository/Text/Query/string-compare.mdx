## Activity Description

Compares two strings.

## Output

–1, 0, or 1

## Settings


* **First String** – The string with which to compare the second string.  
   Example: `%SubjectOld%`
* **Second String** – The string with which to compare the first string.  
   Example: `%SubjectNew%`
