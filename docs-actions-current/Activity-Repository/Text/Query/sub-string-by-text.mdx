## Activity Description

Returns a table or text value of the searched text between a starting and ending point.

## Output

The table or text value.

## Settings


* **Source String** – The text from which to extract the phrase.
* **Start Text** – The starting point of the phrase.
* **End Text** – The ending point of the phrase.
