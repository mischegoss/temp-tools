## Activity Description

Extracts a line from a text, according to its location or a matching keyword.

## Output

The text of the extracted line.

## Settings


* **Text** The text from which to extract the line.  
   Example: `%logs%`
* **Method Type (Specific Line Number/Lines Contain)** – The index of the line to extract/a keyword that the extracted line must contain.
* **Value** – A keyword that the extracted line must contain.
