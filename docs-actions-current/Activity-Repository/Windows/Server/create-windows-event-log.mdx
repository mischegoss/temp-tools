## Activity Description

Creates an entry in the selected Windows event log.

## Output

Success/Failure

## Settings


* **Host Name** – The host name or IP address.
* **User Name** – If required, the username of an account that has read-write access to the host.
* **Password** – If required, the password of the account listed in the User Name field.
* **Event Log Name** – The name of the Windows event log to create.
* **Type/Level** – The log entry's severity.
* **Source** – The log entry's source application.  
   Example: `SRV1`
* **Event Id** – The ID of the new log entry.  
   Example: `456`
* **Category** – The new log entry's category.  
   Example: `Windows`
* **Event** – The log message's text.
