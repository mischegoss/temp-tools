## Activity Description

Performance counters are measurements of system state or activity. They can be part of the operating system or of an individual application. Windows Performance Monitor requests the current value of performance counters at specified time intervals. The possible metrics (categories) are derived from the host on which the activity is executed. If you wish to choose a different category, please copy the metric name and insert it into the Category field.

## Output

The requested counter.

## Settings


* **Check Average** – Performs a single measurement or checks the average for the next selected number of seconds,
* **Seconds** – The number of seconds to check the average for.
* **Category** – The metric to be checked.  
   Example: `LogicalDisk`
* **Counter** – The measurement type.  
   Example: `% Disk Read Time`
* **Instance** – The measurement instance.  
   Example: `_Total`
* **Host Name** – The host name or IP address.
* **User Name** – If required, the username of an account that has read-write access to the host.
* **Password** – If required, the password of the account listed in the User Name field.
