## Activity Description

Returns the current CPU value (usage/availability) of a machine. If there is more than one CPU on the target machine, a table is returned.

## Output

The selected machine's CPU usage/availability value.

## Settings


* **Type** – Select to return the CPU current usage or availability value, as an absolute or a percentage value.
* **Host Name** – The host name or IP address.
* **User Name** – If required, the username of an account that has read-write access to the host.
* **Password** – If required, the password of the account listed in the User Name field.
