## Activity Description

Returns the current disk space value (usage/availability) of a machine.

## Output

The selected machine's disk space usage/availability value.

## Settings


* **Volume** – The disk name. For example, `C:`.
* **Type** – Select to return the disk space current usage or availability value, as an absolute or a percentage value.
* **Host Name** – The host name or IP address.
* **User Name** – If required, the username of an account that has read-write access to the host.
* **Password** – If required, the password of the account listed in the User Name field.
