## Activity Description

Returns the available memory/memory usage of the selected machine (in MB).

## Output

The selected machine's memory usage/availability value.

## Settings


* **Type** – Select to return the selected machine's memory current usage/availability/total value.
* **Host Name** – The host name or IP address.
* **User Name** – If required, the username of an account that has read-write access to the host.
* **Password** – If required, the password of the account listed in the User Name field.
