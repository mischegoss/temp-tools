## Activity Description

Queries a selected entry in the registry according to a selected registry key and a path.

## Output

The value stored in the specified path.

## Settings


* **Path** – The registry path of the queried entry.
* **Registry Key** – The registry key of the queried entry.  
   Example: `HKEY_CLASSES_ROOT`
* **Host Name** – The host name or IP address.
* **User Name** – If required, the username of an account that has read-write access to the host.
* **Password** – If required, the password of the account listed in the User Name field.
