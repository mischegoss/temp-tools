## Activity Description

Extracts a selected compressed file.

## Output

The extracted file.

## Settings


* **Source Archive name** – The location of the compressed file (e.g., `C:\Archive\log.zip`).
* **Archive Password** – The password required in order to extract the compressed file (required only if one was set when compressing the file).
* **Destination ExtractTo** – The destination path of the extracted file (e.g., `C:\unzipped\logs`).
