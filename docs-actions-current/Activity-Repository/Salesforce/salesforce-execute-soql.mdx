## Activity Description

Executes a Salesforce Object Query Language (SOQL) to search your organization's Salesforce data for specific information. SOQL is similar to the SELECT statement in the widely used Structured Query Language (SQL) but is designed specifically for Salesforce data (For more information, visit [Salesforce Developer](https://developer.salesforce.com/docs/atlas.en-us.soql_sosl.meta/soql_sosl/sforce_api_calls_soql.htm)).

## Output

A ResultSet table.

## Settings


* **Module Name** - The name of the Salesforce module in VAR::PRODUCT_FULL.
* **Query** - The query's text (e.g., `SELECT Name,Phone FROM Account`).
