## Activity Description

Creates a new folder on the FTP server.

:::note
This activity should be placed after the Start FTP Session activity and refer to the started session.
:::

## Output

Success/Failure

## Settings


* **FTP Session** – The FTP session in which to create the new folder.
* **Path** – The path where the new folder will be created.  
   Example: `/upload`
