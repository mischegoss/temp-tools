## Activity Description

Checks whether a file exists on the FTP server or not.

:::note
This activity should be placed after the Start FTP Session activity and refer to the started session.
:::

## Output

True/False

## Settings


* **FTP Session** – The FTP session.
* **Path** – The path where the file is located.  
   Example: `/download/log.txt`
