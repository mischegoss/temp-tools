## Activity Description

Deletes a folder from the FTP server.

:::note
This activity should be placed after the Start FTP Create Session activity and refer to the started session.
:::

## Output

Success/Failure

## Settings


* **FTP Session** – The FTP session in which to delete the folder.
* **Path** – The path where the folder is located.  
   Example: `/upload/logs`
