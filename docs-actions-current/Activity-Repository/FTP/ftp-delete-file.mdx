## Activity Description

Deletes a file from the FTP server.

:::note
This activity should be placed after the Start FTP Session activity and refer to the started session.
:::

## Output

Success/Failure

## Settings


* **FTP Session** – The FTP session in which to delete the file.
* **Path** – The path where the file is located.  
   Example: `/upload/logs/log.txt`
