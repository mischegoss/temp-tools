## Activity Description

Lists the folders in a specific path on an FTP server.

:::note
This activity should be placed after the Start FTP Session activity and refer to the started session.
:::

## Output

The array of folders.

## Settings


* **FTP Session** – The FTP session.
* **Path** – The path where the folders are located.  
   Example: `/upload`
