## Activity Description

Renames a folder in the FTP server.

:::note
This activity should be placed after the Start FTP Session activity and refer to the started session.
:::

## Output

Success/Failure

## Settings


* **FTP Session** – The FTP session in which to start the new folder.
* **Source Path** – The path of the folder to rename. 
   Example: `/upload`
* **Target Path** – The path including the new name of the folder.  
   Example: `/upload_2K1`
