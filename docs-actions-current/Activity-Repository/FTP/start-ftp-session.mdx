## Activity Description

Initiates an FTP session.

## Output

Success/Failure

## Settings


* **Use Secure Connection** – Whether to use SFTP.
* **Port** – The port used to access the FTP server.
* **Transfer Mode** – The FTP transfer mode.
* **Session Maximum Time** – The FTP session's maximum time, after which the session is terminated.  
   Example: `00:10:00`
* **Host Name** – The FTP server name.
* **User Name** – A user authorized to access the FTP server.
* **Password** – The password of the user authorized to access the FTP server.
