Incidents activities are used to manage VAR::PRODUCT_FULL incidents. Incident definition and configuration is performed via Actions Incident Configuration.

:::note
Incident activities support constant/changing device, service and classification; i.e., an incident can be ascribed to a permanent device/service and classification, or to variables representing the device/service and classification.
:::
