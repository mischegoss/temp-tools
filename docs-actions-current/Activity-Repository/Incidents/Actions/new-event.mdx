## Activity Description

Simulates an external event generation sent to Resolve Actions. The simulated event goes through the mapping and trigger action cycle.

## Settings


* **Source** – The source from which the new event originates.  
   Example: `%source@example.com%`
* **Destination** – The address to which the new event's alert is sent. This can be any value; the event will be sent to Actions either way.  
   Example: `%dest@example.com%`
* **Subject** – The subject of the event's alert.
* **Body** – The body text of the event's alert.
