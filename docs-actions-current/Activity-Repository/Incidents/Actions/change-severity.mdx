## Activity Description

Changes the severity level of an incident.

## Settings


* **Use Current Incident/Use Another Incident** – Determines whether to change the severity of the incident that triggered the workflow, or another incident.
* **Type: Device/Service** – Determines whether the incident is ascribed to a Device or a Service.
* **Name** – The name of the constant/dynamic incident's Device/Service (if you selected using a constant incident, the list changes according to Type selection).
* **Classification** – The classification of the incident's Device/Service (if you selected using a constant incident, the list changes according to Type selection).
* **Automatically Change the Severity** – Increases or decreases the incident's severity level by 1, regardless of its current level.
* **Or, manually select the Severity** – Sets the incident's severity level to one of the following constant values: Info, Warning, Minor, Major, Critical or a Variable's value (e.g., %Severity%).
