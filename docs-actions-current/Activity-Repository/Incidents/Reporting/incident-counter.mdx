## Activity Description

Returns the number of currently open incidents, according to the selected criteria and time duration.

## Output

The number of open incidents.

## Settings


* **Name** – The name of the incident to retrieve (wild cards are supported).  
   Example: `SQL-SRV`
* **Classification** – The incident's classification (wild cards are supported).  
   Example: `Server Down`
* **Duration in Minutes** – The duration (in minutes) of the incident's activity.
