## Activity Description

Creates a log containing the incident history according to a specified filter.

## Output

The incident report.

## Settings


* **Incidents** – Use the default selected incidents, or use a variable that contains the incidents that will appear in the report.
* **Devices** – Use the default selected devices, or use a variable that contains the devices that will appear in the report.
* **Services** – Use the default selected services, or use a variable that contains the services that will appear in the report.
* **Classification** – Use the default selected classifications, or use a variable that contains the classifications that will appear in the report.
* **States** – Use the default selected states, or use a variable that contains the states that will appear in the report.
* **Event (Contains)** – Events containing the specified string will appear in the report.
* **Severities** – Use the default selected severities, or use a variable that contains the severities that will appear in the report.
* **Time Zone** – Use the default selected time zones, or use a variable that contains the time zones that will appear in the report.
* **Start Date/Current Date** – Uses the current date as the start date.
* **Start Date/Default Values** – Uses the default values as the start date.
* **Start Date/# Days Back** – Uses the given # of days back as the start date.
* **Start Date/Variable** – Uses a variable as the start date.
* **Start Date/Format** – Uses the selected format as the start date.
* **End Date/Current Date** – Uses the current date as the end date.
* **End Date/Default Values** – Uses the default values as the end date.
* **End Date/# Days Back** – Uses the given # of days back as the end date.
* **End Date/Variable** – Uses a variable as the end date.
* **Output/Save Type** – Saves the executed report to file, active logs, or active logs & file.
* **Output/Format** – The format of the executed report.
* **Output/Path** – The path of the file in which to save the report.  
    Example: `C:\Reports\Incidents`
* **Output/Host Name** – The name of the host in which to save the report.
