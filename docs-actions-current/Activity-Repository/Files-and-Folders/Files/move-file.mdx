## Activity Description

Cuts a file from a source location and pastes it into the destination location.

## Output

Success/Failure

## Settings


* **Source**
    * **Path** – The path of the source file.
    * **Host Name** – The host name or IP address.
    * **User Name** – If required, the username of an account that has read-write access to the host.
    * **Password** – If required, the password of the account listed in the User Name field.
* **Destination**
    * **Path** – The path of the destination folder (where the file is pasted).
    * **Host Name** – The host name or IP address.
    * **User Name** – If required, the username of an account that has read-write access to the host.
    * **Password** – If required, the password of the account listed in the User Name field.

:::note
If the file is copied to a remote machine, use a remote location path (e.g., `\\machine name\c$\...`).
:::
