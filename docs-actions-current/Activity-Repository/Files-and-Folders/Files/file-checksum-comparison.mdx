## Activity Description

Compares two files and determines whether they are identical or not.

## Output

True/False.

## Settings


* **Hash Type** – The comparison method.
* **Source Path** – The path where the first file is stored.
* **Host Name** – The host name or IP address.
* **User Name** – If required, the username of an account that has read-write access to the host.
* **Password** – If required, the password of the account listed in the User Name field.
* **Destination Path** – The file where the second file is stored.
* **Host Name** – The host name or IP address.
* **User Name** – If required, the username of an account that has read-write access to the host.
* **Password** – If required, the password of the account listed in the User Name field.
