## Activity Description

Downloads an attachment that was received by email and saves it at the selected server location. The file can then be used by any of the read file activities.

## Output

Success/Failure

## Settings


* **File Name** – The name of the file to download.  
  The value is automatically taken from the Attachment List activity.
  Example: `%getName%`
* **Path** – The path where to store the attachment.
* **Host Name** – The name or IP address of the server.
* **User Name** – If required, the username of an account that has read-write access.
* **Password** – If required, the password of the account.
