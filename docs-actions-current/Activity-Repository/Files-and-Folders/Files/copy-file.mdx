## Activity Description

Copies a specified file from a source location to a destination location.

## Output

Success/Failure

## Settings


* **Source**
    * **Path** – The path of the source file.
    * **Host Name** – The host name or IP address.
    * **User Name** – If required, the username of an account that has read-write access to the host.
    * **Password** – If required, the password of the account listed in the User Name field.
* **Destination**
    * **Path** – The path of the destination folder (should include the file name).
    * **Host Name** – The host name or IP address.
    * **User Name** – If required, the username of an account that has read-write access to the host.
    * **Password** – If required, the password of the account listed in the User Name field.

:::note
If the file is copied to a remote share, use the remote location path convention (e.g., `\\machine name\ShareName`), while the executor device is selected.
:::
