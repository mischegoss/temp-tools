## Activity Description

Retrieves the extension of a selected file.

## Output

The file extension (e.g., in case `%FileName%=log.txt`, the retrieved value would be `txt`).

## Settings


* **Name** – The path where the file is stored (e.g., `C:\%FileName%`).
