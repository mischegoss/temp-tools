## Activity Description

Deletes a folder (and all the files and subfolders that reside in it) in a specified location.

## Output

Success/Failure

## Settings


* **Path** – The path of the deleted folder.
* **Delete Folder Contents Only** – Deletes the content of the folder and not the actual folder.
* **Host Name** – The host name or IP address.
* **User Name** – If required, the username of an account that has read-write access to the host.
* **Password** – If required, the password of the account listed in the User Name field.
