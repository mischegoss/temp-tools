## Activity Description

Sets a folder's permissions.

## Output

Success/Failure

## Settings


* **Path** – The path of the folder.
* **Host Name** – The host name or IP address.
* **User Name** – If required, the username of an account that has read-write access to the host.
* **Password** – If required, the password of the account listed in the User Name field.
* **User/Group Name** – The user or the group to which the permissions are applied.
* **Remove Existing Permissions** – When selected, overrides the existing permissions.

:::note
The following conventions can be used in the user/group name: `DomainName\UserName`, `DomainName\GroupName`, `.\LocalUserName`, `.\LocalGroupName`, `ServerName\LocalUserName`, `ServerName\LocalGroupName`, `LocalUserName`, `LocalGroupName`.
:::
