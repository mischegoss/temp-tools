## Activity Description

Creates a new folder share.

## Output

Success/Failure

## Settings


* **Host Name** – The host name or IP address.
* **User Name** – If required, the username of an account that has read-write access to the host.
* **Password** – If required, the password of the account listed in the User Name field.
* **Folder Path** – The folder to share.
* **Share Name** – The folder share's name.
* **Description** – A description of the folder share.
* **Access Rights** – Comma-separated list of user and group names that can access the new share.
