title: Files and Folders Overview
---

Files and Folders activities are used to create, change, delete or extract data from files and folders.

:::note
When accessing a local machine (or the machine that triggered the workflow), use regular path convention (e.g., `c:\...`). When accessing a remote machine or a shared location use the UNC `\\<Shared Folder Name>`.
:::

:::note
You can use wildcards to indicate unpredictable factors in the file names, e.g., `*.xml`
:::
