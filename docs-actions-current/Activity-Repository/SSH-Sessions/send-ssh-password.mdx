## Activity Description

Sends an encrypted password to an open SSH session. The activity follows the Start SSH Session activity, and is followed by the Send SSH Command and Terminate SSH Session activities.

## Output

Prompt Session

## Settings


* **SSH Session** – The session to which the password is sent (named after the corresponding "Start SSH Session" activity).  
   Example: `StartSSHSession1`
* **Password** – The encrypted password.
