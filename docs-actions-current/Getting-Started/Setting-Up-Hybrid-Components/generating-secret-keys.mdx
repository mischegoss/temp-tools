title: Generating Secret Keys
---

After taking note of your Remote Access Key, you must generate an event key for API access.  
1. **Event Key** - for use with [webhooks](../../Product-Navigation/Configuration/Integrations-and-Modules/Integration-Modules/using-webhooks-to-forward-events.mdx).
Multiple Event keys can be generated. 


Take these steps to generate a secret key:

1. Click the hamburger menu at the top navigation bar and select **Configuration** > **API Access**.
2. In the **Secret Keys** section, click the **plus (+) sign**. 
3. Within the **Secret Keys – New secret Key** sidebar, enter the following: 
    * **Name** -  enter a name that will help you identify the key.  
    Example: Austin Hybrid Executor
    * **Type** - select the desired key type:
        * **Event** to generate an API access key to use with webhooks.
4. Click **Save**.
5. From the **Generate Secret Key** dialog box that appears, use the copy function next to the **Value** field to copy your secret key to your clipboard.
    
:::caution
The value is only presented once. If you fail to copy the value, you must regenerate the key.
:::
