The VAR::PRODUCT_FULL Hybrid Installer is a standard Windows installer. Take the following steps to run the Resolve Actions Remote Executor Installer:

1. Open the hamburger menu from the top navigation bar and click **Configuration > Download Remote Installer**.
2. Copy the Installer to the machine that will be running the on-premises components of Actions.
3. Double-click the Installer file to start it.
4. In the **User Account Control** screen presented by Windows, click **Yes** to agree to continue with the installation.
5. On the **License Agreement** screen, read the End User License,  then click **I Agree**.
6. On the **Tenant Information** screen, paste the **Remote Access Key** acquired in [Acquiring API Access Details](../../Getting-Started/Setting-Up-Hybrid-Components/acquiring-api-access-details.mdx).
7.  Select a new installation folder on the  **Choose Install Location** screen or click **Install to use the default folder (recommended)**.
8.  On the Installation Completed screen, choose **Start the Resolve Actions Manager Service** and
click **Finish**.

To complete configuration, see [Configuring the Setup](./Configuring-the-Setup.mdx).
