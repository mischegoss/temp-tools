title: Acquiring API Access Details
---

After you log in, you need to take note of your Remote Access Key. You will need this information later when running the VAR::PRODUCT_FULL Remote Agent Installer.

Follow these steps to obtain your Remote Access Key:

* Click the hamburger icon in the top menu.
* Navigate to **Configuration** and click **API Access**.
* Find the **Remote Access Key** in the **Tenant Details** section.
* Expand the menu to the right to show the **Copy to Clipboard** function.
* Use the **Copy to Clipboard** function to copy the **Remote Access Key**.
