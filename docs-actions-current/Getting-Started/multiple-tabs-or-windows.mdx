title: Multiple Tabs or Windows
---

You can open multiple sessions in separate tabs or windows if needed. This is helpful when working in the Workflow Designer and you want to access another navigation item without leaving the Designer.

To open another session:

- Copy the URL you used to log in.
- Open a new browser tab or window.
- Paste the copied URL into the new tab or window.

The session will open at the main navigation menu, and you won't need to log in again.
