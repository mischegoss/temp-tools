title: Accessing the Self Service Portal
---

To display the Self Service Portal, open the main menu and select **Home Page > Self Service Portal**.

:::note
Any role can see the Self Service Portal. For details about permissions, see [Access Control](../../../Product-Navigation/Configuration/Integrations-and-Modules/access-control-on-integrations-and-modules.mdx).
:::
