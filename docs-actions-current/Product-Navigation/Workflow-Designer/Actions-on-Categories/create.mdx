Take these steps to create a new category:

1. Open the Activity Designer.
2. In the left navigation bar, click **+ New Category**.
3. In the **New Category** dialog box that appears, enter the category name.

You can also create a new category from the New Activity Modal.

## Creating a Subcategory

Subcategories are organized as folders within existing folders. Follow these steps after creating a category:

1. Open the Activity Designer.
2. In the left navigation bar, click **+ New Category**.
3. In the **Create Category** dialog box: 
   1. Enter the *Category Name.*
   2. In *Create as a sub-folder of:* select the name of the parent folder.
