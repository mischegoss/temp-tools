## Activity Description

Terminates the current branch in the current workflow. This is typically used when two parallel workflows are run via one workflow, or to return a value from the workflow to an VAR::PRODUCT_FULL Service request.

## Settings


* **Reason** - The reason for terminating the branch. The reason is registered to the workflow's log or returned to the Actions Web Service.  
  Example: `False Alert`
