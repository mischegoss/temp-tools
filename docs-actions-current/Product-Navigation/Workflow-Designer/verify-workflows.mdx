## Verifying Workflows {#UUID-bbf488df-afef-0d5d-41df-dc3a1a921fd1}

The Verify function checks a workflow for valid configuration, ensuring all components (e.g., activity settings) are properly defined. Run verification before saving or executing the workflow.

To verify a workflow:

*   Click the three-dot menu on the left side of the [workflow tab](./workflow-dashboard.mdx) and select **Verify**.  
    A confirmation message will appear on a valid workflow.  
    An Error Log will appear at the bottom of the screen for an invalid workflow. Each row of the Log displays the name of the component that failed verification and a description of the error involved.
