title: Notes
---
The **Notes** tab of the Activity Details dialog allows you to enter comments about the activity. Notes can include a description, its role in the workflow, or any other helpful context. 

**Recommendation:** Add a note for each activity to improve workflow clarity and collaboration.

To add or update a note:

1. Enter your text in the **Notes** field.
2. Click **Save.**

All notes for a workflow appear at the bottom of the **Documentation** dialog.
