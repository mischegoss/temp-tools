## Adding Tags to a Workflow 

Tags help organize and find workflows. You can use existing tags or create new ones.

To create a new tag:

1.  On the **Create New** screen, click **Add** next to **Tags**.  
2.  Enter the tag name and click **Add**.  
3.  Repeat as needed, then click **Save**.  
    :::note
    Tags can be removed using the X icon or the **Clear all** button. You can also edit tags when saving the workflow.
