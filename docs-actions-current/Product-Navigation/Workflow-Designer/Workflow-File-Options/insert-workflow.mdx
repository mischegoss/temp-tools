### Inserting Existing Workflows

If the imported workflow already exists choose one of the following:

1.  **Use existing** - Skip import and keep existing copy.
2.  **Keep both** - Import under a new name (ending in `_Imported`).
3.  **Overwrite** - Replace existing workflow.
