title: Open Template
---
Templates are customizable, pre-built workflows fro common use cases. 

To open a template:

1.  Click the **Templates** tab on the **Welcome** screen, or click **Search Templates** under **Create New**.
    ![Workflow Designer Templates](img/wd-workflow-templates.png)
2.  Filter templates:
    *   **By Tag:** Select tags in the **Tags** field. See [Adding Tags](./add-tags.mdx).
    *   **By Search Term:** Enter keywords in the **Search** field. 
        
    The following example shows a Templates list that is filtered according to both the Active Directory tag and the search term _user_:
    
    ![Workflow Designer Templates Menu Filtered](img/wd-workflow-templates-filtered.png)
3.  Select a template to open it in the Designer.
