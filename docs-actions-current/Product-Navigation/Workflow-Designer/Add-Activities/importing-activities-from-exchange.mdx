title: Importing Activities from the Automation Exchange
---
The [Resolve Automation Exchange](https://exchange.resolve.io/) is a web-based library of pre-built automations and integrations, designed to accelerate automation without scripting or coding. You can browse, search, download and import these assets into your environment.

To download and use an activity:

1. Open the [VAR::EXCHANGE](https://exchange.resolve.io/).
2. Click **Resolve Actions**.
3. Find the activity:
   1. Browse the list,
   2. Use the *Search Automation Exchange* box. 
4. Download the activity:
    * Clicking the Download icon in the *Actions* column left of the activity name;
    * Clicking on the activity name and then the **Download** button in the package detail window that appears.
5. [Import the activity](../../Activity-Designer/Importing-Activities.mdx) into VAR::PRODUCT.
6. Add it to your workflow as usual.
