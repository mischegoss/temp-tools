When selecting *Start from Scratch*, you will be prompted to fill in:
1. *Activity Name* - give the activity a unique name
2. Select *Executor* or *Custom Integration* (note: the default is Executor)
3. *Category* - choose to add your activity to a selected category or *+Add Category* to create a new category.
4. *Description* - enter a description for the activity.
5. Click *Save & Continue*. 

### Selecting the Activity Language

Once you enter the Activity Designer, you will see a window with three tabs:
- Json
- Code
- Help
  
For more on JSON, please see our [JSON Overview](./json.mdx)

Under the code tab, you have the option of selecting C#, Visual Basic, or Python.
