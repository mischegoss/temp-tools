title: Introduction to the Self Service Designer
---

The Self Service Designer is a simple and efficient tool that enables you to design forms by integrating workflows while exposing relevant automated tasks and processes to your end-users.

Self Service Automation can be achieved by following these steps:

1. Create a Workflow with the following:
    *   A Workflow activity containing [Self Service variables](./Creating-a-Self-Service-Workflow.mdx). 
    *   A [Self Service Response Activity](./Creating-a-Self-Service-Workflow.mdx). 
2. Using the Self Service Designer, create a related [Self Service Form](./Creating-a-Self-Service-Form.mdx)
3. Add additional [Form fields](./Form-Fields.mdx).
3. Assign form [Permissions](./Setting-Permissions-on-a-Form.mdx).
4. Activate the Form with the [Enable Checkbox](./Enabling-a-Form.mdx).

Once these steps are completed, the form will be available to all permitted users in the Self Service Portal.

To open the Self Service Designer:

1. Open the Main Menu in the top left corner.
2. Under the **Builder** section, click **Self Service Designer**.

:::note 

To access the Self Service Designer, you need one of the following [user roles](../Configuration/User-Management/managing-login-users.mdx)

*   Administrator
*   Developer
*   Workflow Editor
*   Specialist
*   Operator
:::
