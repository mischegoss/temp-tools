title: Generating Blocks From a Workflow
---

To automatically create blocks (fields) from a Workflow:

1. In the **Control Panel**, in **Workflow name**, select the workflow to base the form on.
2. Click **Generate Blocks**.  
   - In the Workflow, Self Service variables appear as *Text fields* and are automatically added to the left column of the Form. The field label is the variable name.

:::note
You can add new fields, remove generated fields, or move fields around. Controls are generated as Text by default; to change the default, delete the current control and drag and drop a new control.
:::
