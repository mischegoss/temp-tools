title: Entering Form Metadata
---

Start creating a Form by adding the following metadata

*   Name (required)
*   Folder (required). Click **Add Folder** to create a folder without leaving the **Create Form** wizard.
*   Description
*   Tags

:::note
A description is particularly important because it will allow your users to know the purpose of the Form and when to use it.
:::
