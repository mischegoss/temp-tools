title: Manually Adding Fields to a Form
---

If you don't want to use the Generate Blocks feature, you can manually drag desired fields to the Form. Manually adding fields can assist in creating a better user experience. You can add titles and organize fields into relevant groups. 

1. From the **Control Panel**, click and hold the desired field type and drag it onto the Form.
2. You can also drag fields to either column in the Form.
