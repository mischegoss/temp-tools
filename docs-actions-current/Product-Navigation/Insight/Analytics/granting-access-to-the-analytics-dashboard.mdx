title: Granting Access to the Analytics Dashboard
---

In a fresh installation, you will be able to access the **Analytics** dashboard if you have any of the following [user roles](../../Configuration/User-Management/managing-login-users.mdx):

* Administrator
* Developer
* Workflow Editor
* Specialist
* Operator
* Advanced User
* Analytics User

:::note The "Analytics user" role

The **Analytics User** user role is a special-purpose role whose main purpose is to allow access to the **Analytics** dashboard. Holders of this role have full view and edit access to the dashboard.

These are the menus that the role has full access to:

* Home Page
    * VAR::PRODUCT_FULL LIVE
    * Self Service Portal
* Insight
    * Audit Trail
    * Incident History
    * Analytics
:::
