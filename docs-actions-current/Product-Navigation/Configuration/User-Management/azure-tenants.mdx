VAR::PRODUCT_FULL enables the synchronization of users and groups from Azure Active Directory (AD).

The **User Management > Azure Tenants** table is a list of all Azure tenants currently synced with your VAR::PRODUCT installation. It is populated with tenants that are synced using the [Azure AD Users Synchronization Activity](../../../Activity-Repository/Azure/Azure-Overview.mdx) activity, displayed with their Tenant Name, Domain Name, and App Client ID.

See how to [sync Azure AD users](../../../Product-Navigation/Repository/Recipients/Azure-AD-User-Synchronization.mdx) with VAR::PRODUCT.
