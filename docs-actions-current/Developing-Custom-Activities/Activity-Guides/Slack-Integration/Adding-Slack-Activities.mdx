title: Adding Slack Activities
---
Once you have set up your Slack Bot and added all of the [necessary configuration](./Configuring-the-Slack-Integration.mdx), you can [create a new workflow](../../../Building-Your-Workflow/new-workflows.mdx) and add Slack Activities.

In a new Workflow canvas, you can add any of the six Slack activities by dragging them from the left side menu, or clicking the + sign and searching for Slack. 

- [Slack Archive Channel](../../../Activity-Repository/Slack/slack-archive-channel.mdx)
- [Slack Create Channel](../../../Activity-Repository/Slack/slack-create-channel.mdx)
- [Slack Invite to Channel](../../../Activity-Repository/Slack/slack-invite-to-channel.mdx)
- [Slack Kick from Channel](../../../Activity-Repository/Slack/slack-kick-from-channel.mdx)
- [Slack List Users](../../../Activity-Repository/Slack/slack-list-users.mdx)
- [Slack Send Message](../../../Activity-Repository/Slack/slack-send-message.mdx)

When adding an activity, the tokens you saved in Global Variables will automatically populate the activity. Follow each activity's guide for more information about unique information that must be added to the activity.
