* **Description:** Masked text input for sensitive credentials  
* **Base Type:** control  
* **Data Type:** String (Password)  
* **Example Values:** `\*\*\*\*\*\*\*\*` (masked)  
* **Common Use Cases:** Passwords, API keys, Secret tokens, Credentials  
* **Optional Properties:**  
  * value (String): Default value  
  * required (Boolean): If true, the field must be filled. Default is false

**JSON Example:**  
```
{
  "key": "secretKey",
  "label": "Secret Key",
  "baseType": "control",
  "controlType": "password",
  "value": ""
}
```
